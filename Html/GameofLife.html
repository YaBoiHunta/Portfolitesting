This is the start of the Game of Life.

This is a game that is based on John Conway's game of life. But with my own little spin and creative twist on it.

The rules are simple.

  • Any live cell with fewer than two live neighbors dies, as if by under population.
  • Any live cell with two or three live neighbors lives on to the next generation.
  • Any live cell with more than three live neighbors dies, as if by overpopulation.
  • Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.

The game is played on a grid of cells. Each cell can be either alive or dead.
